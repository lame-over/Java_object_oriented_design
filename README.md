# LMS - Система управління навчанням

Імітаційний прототип об'єктно-орієнтованої системи управління навчанням (Learning Management System) розроблений на Java.

## Огляд проєкту

Проєкт реалізує функціональність системи LMS з такими основними компонентами:

- **Студент** - користувач, який записується на курси, подає роботи, отримує оцінки та може подавати апеляції
- **Викладач** - користувач, який створює курси, завдання, перевіряє роботи студентів, ставить оцінки
- **Адміністратор** - користувач, який керує системою, налаштовує курси, встановлює дедлайни, управляє доступами

## Основні класи та їхні функції

### Training_Participant (Абстрактний клас)
Інкапсулює спільну поведінку всіх учасників навчального процесу:
- `get_profile()` - отримати профіль користувача
- `authorization()` - перевірити авторизацію

### Educational_schedule (Інтерфейс)
Контракт для роботи з освітнім розкладом:
- `get_schedule()` - отримати розклад занять
- `deadline_check()` - перевірити дедлайни
- `show_curriculum()` - показати навчальний план

### Student (Клас студента)
Функціональність:
- Запис на курси та оплата
- Подання робіт до завдань
- Перегляд оцінок
- Подача апеляцій
- Отримання академічного сертифіката

### Teacher (Клас викладача)
Функціональність:
- Створення нових курсів та завдань
- Перевірка робіт студентів
- Перевірка на плагіат
- Ставлення оцінок
- Розгляд апеляцій студентів

### Administrator (Клас адміністратора)
Функціональність:
- Управління курсами
- Налаштування дедлайнів
- Управління доступами користувачів
- Перегляд статистики системи

### Course (Клас курсу)
Представляє навчальний курс з кодом, назвою, описом, кількістю кредитів та ціною.

### Exercise (Клас завдання)
Представляє завдання у курсі з дедлайном, максимальною оцінкою та статусом.

### Appeal (Клас апеляції)
Представляє апеляцію студента на оцінку з можливістю прийняття або відхилення.

## Компіляція

Для компіляції Java-файлів виконайте команду:

```bash
cd src
javac *.java
```

Або використайте скрипт:

```bash
bash run.sh
```

## Запуск програми

Для запуску симуляції системи виконайте команду:

```bash
java -cp src Main
```

Або за допомогою скрипту:

```bash
bash run.sh
```

## Очікуваний результат

Програма виведе симуляцію роботи LMS-системи з наступними кроками:

1. Налаштування системи адміністратором
2. Створення курсів та завдань
3. Реєстрація студентів та оплата курсів
4. Подання робіт студентів
5. Перевірка робіт викладачем
6. Перегляд оцінок студентами
7. Подача та розгляд апеляцій
8. Перегляд розкладів та дедлайнів
9. Показ статистики системи
10. Видача академічних сертифікатів
11. Показ навчальних планів

## Принципи об'єктно-орієнтованого програмування, використані у проєкті

- **Інкапсуляція** - виховуються приватні атрибути та публічні методи доступу
- **Спадкування** - класи Student, Teacher, Administrator спадкують від Training_Participant
- **Поліморфізм** - всі класи реалізують методи интерфейсу Educational_schedule
- **Абстракція** - використовується абстрактний клас та інтерфейси для визначення контрактів
- **Композиція** - Appeal містить посилання на Student та Exercise
- **Агрегація** - Course містить колекцію Student та Exercise
