# Система управління навчанням (LMS)

Імітаційний прототип системи управління навчанням, що демонструє взаємодію між студентами, викладачами та адміністраторами.

## Компіляція

Для компіляції проєкту виконайте наступну команду з кореневої директорії проєкту:

```bash
javac -d bin src/*.java
```

## Запуск

Після компіляції, запустіть програму наступною командою:

```bash
java -cp bin LMSDemo
```

## Структура проєкту

- `src/` - вихідний код програми
- `docs/` - документація проєкту (діаграми UML та вимоги)
- `bin/` - скомпільовані class-файли (створюється при компіляції)

## Функціональність

Прототип демонструє основні функції системи:
- Реєстрація та авторизація користувачів
- Управління курсами
- Подання та перевірка робіт
- Оцінювання
- Процес апеляції
